Coin Tracker

JavaScript blockchain implementation

Project Summary

Coin Tracker is a JavaScript blockchain implementation with an AngularJS front end that allows the user to send coins to an address, mine blocks and keep a record of each ledger in the chain.

Creative Process

This project was my first time using AngularJS. Even though I had previous experience using frameworks such as Vue and React, it was a challenge adjusting to Angular's file structure and it felt like accomplishing the simplest tasks felt pretty complicated, but after the blockchain service itself was built, finishing the front end to display and manipulate the data was fairly easy.

Results

I built this project in an effort to gain a better understanding of blockchain technology and how it works on a technological level. Thanks to the knowledge gained from this process, I plan to build fully decentralized apps on the Etherium blockchain in an effort to liberate and empower artists.
