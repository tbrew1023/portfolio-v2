Project Summary

Jiyoun Jun is a Chicago based UX Designer, and visual artist. I was comissioned to help build a website that showcases the full spectrum of her talents.

Creative Process

The site was mapped out with full-page auto scrolling in mind, designating a specific category for each row, allowing the user to view more prjects in that category as they scroll horizontally.

The site maintains it's dark theme and monochrome color palette from the original site seen below. The landing page utilizes p5.js effects to draw onto the screen as the user moves the cursor.

Results

This project was a great introduction into embedding p5/three.js canvases into an existing project for processing 2d and 3d graphics, allowing for interesting user interactions. This project also utilizes a custom client-facing CMS back-end tool to keep projects up to date.
