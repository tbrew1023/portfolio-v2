Nihal.tv

Full-stack FIlm Portfolio

Project Summary

Nihal.tv is a film portfolio showcasing the work of Nihal Dantluri, a Chicago based filmmaker and director of photography.

Creative Process

Nihal.tv was originally meant to be built with Webflow, a powerful CMS tool for dynamic web pages. Due to limitations with Webflow's front end tools, I decided to build Nihal.tv from scratch, using VueJS on the front end and Firebase on the back end. Nihal.tv is comprised of 2 halves, the first being a Firebase powered custom CMS tool that allows the client to upload and remove projects, and the other is a fullscreen carousel of the client's work with animations and interaction meant for public viewing. The custom CMS tool can be seen below.

Results

I learned a lot about full-stack development through this project. It allowed me to play around with REST APIs, asynchronous storage, as well as authentication. VueJS and Firebase is quite a harmonious combination and I had a lot of fun crafting together this portfolio as it's very visual and allowed me to experiment with different interactions, animations and effects.
