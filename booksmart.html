Booksmart

Bookmark management chrome extention

Project Summary

Booksmart is a Chrome extension that aims to help users manage their browser bookmarks by introducing a customizable folder interface with grouped URLs.

Creative Process

Booksmart was built using HTML, SASS, and JQuery. Assets such as the folder icon were made as vector images using Figma. The draggable interface is powered by JQuery UI and Packery. Loading multiple tabs at once was a big challenge since the normal window.open function only allows for one tab to be opened even for multiple calls. This is due to Chrome's popup-blocking, but I was able to accomplish loading multiple tabs by adjusting the permissions in the manifest file and using the chrome.tabs.create method.

Results

Booksmart is my first ever chrome extension I've built, and I didn't expect for it to turn out to be such a useful tool. While building it I tried to keep in mind what features would help streamline my workflow, and I think the result is a tool that will help me and hopefully help other students manage their web experience and maximize productivity.
