Raster.video

Full-stack CMS for Visual Art

Project Summary

Raster.video is a CMS tool built to allow clients to upload, edit, and delete projects from their portfolios.

Creative Process

Due to limitations with the design tools for tools like Webflow, Squarespace, and Wix, I built a custom CMS for clients to upload their projects which can then be fetched and rendered in a client-facing site without any design restrictions. Clients are able to upload image/video files, update project information, remove files, and change the bio that appears in their contact page. Basic templates for new users are in development, the Showcase feature will allow users to create and upload their own templates using a built-in editor.

Results

This project helped me to gain fluency with Firebase storage, cloud functions, and server-side image compression. I hope to scale up Raster.video into a tool that can be used to create and upload templates for building any website.
